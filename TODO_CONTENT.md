# Inhalte & To-Do für den Kunden

Damit die Website live gehen kann, benötigen wir folgende echte Inhalte:

## 1. Bilder (Hochauflösend)
- [ ] **Hero-Bild Startseite**: Einladendes, stimmungsvolles Bild vom Laden oder Produkten (Querformat).
- [ ] **Eva Vogel**: Porträtfoto für die "Über uns" Seite.
- [ ] **Spezialitäten**:
    - [ ] Antipasti Teller / Glas
    - [ ] Pasta Packung / Gericht
    - [ ] Dolci (Cantuccini o.ä.)
    - [ ] Weinflasche / Glas
- [ ] **Geschenkkörbe**: Fotos der 3 Beispiel-Grössen (S, M, L).
- [ ] **Catering**: Foto einer Apéro-Platte oder Buffet.

## 2. Texte & Daten
- [ ] **Testimonials**: 2-3 echte Kundenstimmen (kurz, mit Vorname).
- [ ] **Produkte**: Prüfen Sie die Liste in `src/content/products.ts` auf Vollständigkeit und korrekte Preise/Beschreibung.
- [ ] **Impressum & Datenschutz**: Rechtliche Texte müssen vom Anwalt oder Treuhänder geprüft/geliefert werden.

## 3. Technisches
- [ ] **Resend.com Account**: Für den E-Mail-Versand benötigen wir einen API Key (oder SMTP Zugangsdaten, falls anderer Provider gewünscht).
- [ ] **Domain DNS**: Anpassung der DNS-Records für Vercel (A Record / CNAME) beim Go-Live.
